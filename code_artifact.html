遊戲核心機制
歡迎來到健康大富翁！
1. 輪到玩家時，按下按鈕擲出 1~6 點骰子。
2. 系統會依據您挑戰的目標格主題，跳出對應的專業性教育知識問答。
3. 答對題目：可前進骰子的全額步數，獲得元氣積分！
⏳
懲罰與勝負條件
1. 答錯無條件捨去：答錯者移動步數折半並捨去小數點（例如骰出3步，折半後只走1步）。累計答錯滿 3 次將被強制關進「🏥 隔離病房」停賽一輪！
2. 先滿 300 分即獲勝：率先達到 300 分的玩家獲得冠軍！若有人元氣低於 0 分破產，遊戲也會提前結束並進行排行。
